# SymptomAnalyzer

A diagnostic tool that allows users to input symptoms and receive potential conditions and recommendations for further action, powered by advanced natural language processing.

## Features

- Input symptoms through a user-friendly interface
- Receive potential conditions based on symptom analysis
- Get recommendations for further medical action
- Powered by Bio-Medical-MultiModal-Llama-3-8B-V1, a specialized large language model for biomedical applications

## Technology Stack

- Python (Flask backend)
- HTML/CSS/JavaScript (Frontend)
- Bio-Medical-MultiModal-Llama-3-8B-V1 model integration

## Installation

1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run the application: `python app.py`

## Usage

1. Enter your symptoms in the input field
2. Click "Analyze" to receive potential conditions and recommendations
3. Consult with a healthcare professional for accurate diagnosis
